ℹ️ About Project Shield

CFSS Global Internship 2026

🎯 Project Overview

Project Shield is an AI-Powered Security Analytics Dashboard built for the CFSS Global Internship 2026. It ingests security logs, applies machine learning models to detect anomalies, and generates real-time threat scores to identify potential cyberattacks.

🤖 AI/ML Architecture
🔹 Isolation Forest — Anomaly detection
🔹 Random Forest — Multi-class attack classification
🔹 Threat Scoring — 0–100 composite risk metric
🔹 Dataset — NSL-KDD style (860 samples)
🏗️ Tech Stack
⚡ Frontend — HTML5, Vanilla JS, CSS3, Chart.js
⚡ Backend — Python Flask REST API
⚡ Auth — JWT (flask-jwt-extended)
⚡ ML — scikit-learn, NumPy, Pandas
⚡ Security — CORS, Input validation, OWASP Top 10
🛡️ Attack Categories
Normal  Legitimate traffic
Probe  Network scanning / reconnaissance
DoS  Denial of Service floods
R2L  Remote-to-Local intrusion
U2R  User-to-Root privilege escalation
